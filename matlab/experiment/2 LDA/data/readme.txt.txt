This folder contains two preprocessed data sets:

1. Colon cancer dataset
2. Leukemia dataset

The original data is in R format from the Dettling (2004) paper, link: http://stat.ethz.ch/~dettling/bagboost.html
The data is converted to the Matlab readable, also downloadable from: http://www.stat.cmu.edu/~jiashun/Research/software/HCClassification
